# Bobo Bot

Bobo Bot is a Telegram bot that confirms user phone numbers by sending a POST request to a specified API endpoint.

# Configuration

The bot expects the following environment variables to be set:

* `BOT_TOKEN`: the Telegram bot token
* `API_URL`: the URL of the API endpoint to send the POST request to

# Run bot

To run the bot, execute the following command in your terminal:

`node index.js`
